Hallo nama saya adalah ahmad fauzan panggilan nama saya adalah fauzan arti dari nama fauzan adalah kesuksesan sedangkan ahmad adalah nama dari junjungan saya yaitu Nabi Muhammad SAW saya memiliki Hobi berenang.Saya memiliki cita cita yaitu menjadi seorang IT programming Di tahun ini saya adalah siswa SMA,SMA saya terletak di daerah manukan Surabaya jawa timur,rencana saya setelah lulus dari SMA saya ingin bekerja dulu sambil mempelajari tentang program. saya ingin bekerja sebagai programming setelah lulus dari SMA,Oleh karena itu sebelum saya bekerja saya akaan mengikuti kursus IT dan mendapatkan sertifikat kerja lalu setelah saya menabung dari hasil kerjaan saya saya akan membayar kuliah saya dengan hasil kerjaan saya.

Alasan saya ingin menjadi programmer dikarenakan saya tertarik dengan bahasa programnya dan proses pembuatan programnya.saya di SMA memiliki banyak teman yang mana teman tersebut adalah teman sekelas.Sebenarnya saya memiliki teman yang dari kelas lain tetapi tidak banyak,dan saya masih kelas 10,dikarenakan kurikulum ditahun ini adalah kurikulum merdeka dikarenakan kurikulum merdeka ditahun ini kelas 10 SMA tidak ada yang namanya pilih jurusan tetapi di saat saya kelas 11 saya baru akan diperintahkan untuk memilih jurusan.
